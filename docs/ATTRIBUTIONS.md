# ATTRIBUTIONS

Every third-party asset in this repository must have an entry in this file. An
asset without an entry is not allowed in the project and must be deleted.

## Required fields

Each entry records:

- Asset name and stable provider identifier
- Direct source page URL
- Author or credited organization
- License and license URL
- Date pulled in `YYYY-MM-DD` format
- Repository paths and intended Lanka usage

Entries generated by the M1 material pipeline are enclosed in stable HTML
markers. Do not edit or remove those markers; the M2 attribution validator will
use them.
